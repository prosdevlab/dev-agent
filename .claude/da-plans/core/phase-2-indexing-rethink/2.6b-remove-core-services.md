# Part 2.6b: Remove Core Services, Subagent GitHub Module, Types, Update Exports

See [overview.md](overview.md) for architecture context.

## Summary

After Part 2.6a removes the consumers of git/github code (adapters, CLI
commands), this part removes the underlying implementations: `GitHistoryService`,
`GitHubService`, `GitIndexer`, `LocalGitExtractor` from core, the entire
`packages/subagents/src/github/` module, `GitHubIndexer`, planner GitHub utils,
and the `@prosdevlab/dev-agent-types/github` types. Update all exports and check
for dangling references.
