## Packages

### @prosdevlab/dev-agent-core

The foundation layer providing:

- **Scanner** вҖ” Multi-language AST parsing:
  - TypeScript/JavaScript via ts-morph
  - Go via tree-sitter WASM
  - Markdown via remark
- **Indexer** вҖ” Converts code components to vector embeddings
- **Vector Storage** вҖ” LanceDB for fast similarity search
- **GitHub Integration** вҖ” Fetches and indexes issues/PRs via GitHub CLI

### @prosdevlab/dev-agent-cli

Command-line interface:

```bash
dev index .              # Index repository
dev mcp install          # Install MCP integration
dev github index             # Index GitHub issues/PRs
dev status               # Check indexing status
```

### @prosdevlab/dev-agent-mcp

MCP server with tool adapters:

- **AdapterRegistry** вҖ” Manages tool registration and execution
- **Built-in Adapters** вҖ” Search, Plan, Explore, GitHub, Status, Health
- **Rate Limiting** вҖ” Token bucket algorithm (100 req burst)
- **Retry Logic** вҖ” Exponential backoff with jitter

### @prosdevlab/dev-agent-subagents

Specialized AI agents:

- **PlannerAgent** вҖ” Generates implementation plans from issues
- **ExplorerAgent** вҖ” Code pattern analysis and relationship mapping
- **PRAgent** вҖ” PR management and review assistance
- **Coordinator** вҖ” Routes tasks to appropriate agents

### @prosdevlab/kero

Centralized logging:

- Multiple log levels (debug, info, warn, error)
- Console and file transports
- JSON and pretty formatters
- Structured metadata

## Supported Languages

| Language | Scanner | Features |
|----------|---------|----------|

## Storage

All data is stored locally in `~/.dev-agent/`:

```
~/.dev-agent/
в”ңв”Җв”Җ indexes/
в”Ӯ   в””в”Җв”Җ <repo-hash>/
в”Ӯ       в”ңв”Җв”Җ code.lance/        # Code vector index
в”Ӯ       в””в”Җв”Җ metadata.json      # Index metadata
в”ңв”Җв”Җ github/
в”Ӯ   в””в”Җв”Җ <repo-hash>/
в”Ӯ       в”ңв”Җв”Җ issues.lance/      # Issues vector index
в”Ӯ       в””в”Җв”Җ prs.lance/         # PRs vector index
в””в”Җв”Җ config.json                # Global config (optional)
```

## Security

- **100% Local** вҖ” No data sent to external services
- **No Cloud** вҖ” All processing happens on your machine
- **Embeddings** вҖ” Generated locally via @xenova/transformers
- **GitHub Data** вҖ” Fetched via authenticated GitHub CLI
